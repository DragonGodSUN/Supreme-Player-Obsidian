## 快速开始

### 启用插件

1. 打开 Obsidian 设置
2. 进入**第三方插件**，确定**安全模式**关闭
3. 找到**打开插件文件夹**，通常会打开**你的仓库名\.obsidian\plugins**
4. 将本插件打包成文件夹丢进去，注意里面必须包含**main.js、manifest.json、styles.css**三个关键文件
5. 重新加载插件，找到 **Supreme Player** 并启用

### 验证运行

启用后，状态栏底部会显示：
```
⭐ Lv.0 星语者 | 📊 0 | 🌟 0
```

**所有功能都从这里开始。**

## 状态栏面板

将鼠标悬浮在状态栏文字上，约 0.5 秒后弹出快捷面板。

点击面板底部的按钮，可快速进入：

- **⛲ 许愿** → 打开许愿池
- **🎴 商城** → 打开道具商城
- **📦 背包** → 查看和使用物品

---

## 玩家面板

点击状态栏的**「📊 积分」**区域，打开玩家面板。

### 面板功能

| 区域 | 说明 |
|------|------|
| 积分/等级/愿星 | 当前数据一览 |
| 距下一级 | 还需多少积分升级 |
| 货币 | 稀有/传奇道具卡数量 |
| Buff | 当前生效的增益效果 |
| 操作按钮 | 打卡、许愿、商城、背包 |

### 每日打卡

点击面板中的**「📅 每日打卡」**：

1. 自动读取今日日记
2. 计算已完成任务的积分
3. 显示进度条和完成情况
4. 点击**「✅ 确认打卡」**完成

**任务积分：**

| 任务类型 | 数量 | 单项积分 | 最大积分 |
|---------|------|---------|---------|
| ★ 每日三件事 | 3 | 100 | 300 |
| ● 习惯打卡 | 3 | 50 | 150 |
| ● 额外任务 | 2 | 50 | 100 |
| ⏰ 番茄钟 | 6 | 50 | 300 |

## 许愿池

### 入口

悬浮面板点击**「⛲ 许愿」**，或玩家面板点击**「⛲ 前往许愿」**。

### 使用流程

1. 点击**「🌟 许下新愿望」**创建愿望
2. 在进行中的愿望上点击**「✨ 投入」**消耗愿星
3. 每次投入增加 10% 进度
4. 进度满 100% 后点击**「🌟 完成许愿」**

### 奖励

- 获得 500 积分
- 获得"好运"Buff（24 小时）

## 完美打卡

当日完成所有基础任务（三件事 + 习惯打卡），打卡时触发完美打卡。

获得**超级大钻石 💎**，使用后：
- 获得 1-5 颗愿星（随机）
- **20% 概率**额外获得 5 颗愿星
- **1% 概率**额外获得 100 颗愿星
- 获得"超好运"Buff（1 小时）

## 进阶用法

### 命令面板

按 `Ctrl+P` 打开命令面板，搜索以下命令直接执行：

| 命令 | 功能 | 说明 |
|------|------|------|
| 每日打卡 | 执行今日打卡 | 等同于面板中的打卡按钮 |
| 查看当前状态 | 打开玩家面板 | 等同于点击状态栏积分区域 |
| 同步账户信息 | 同步数据到日记 | 将当前积分、愿星等数据写入今日日记 |
| 许下愿望 | 创建新愿望 | 直接跳过许愿池进入创建界面 |
| 查看许愿池 | 管理愿望 | 查看所有进行中和已完成的愿望 |
| 打开商城 | 浏览购买商品 | 查看和购买商城商品 |
| 查看背包 | 使用背包物品 | 查看和使用已购买的物品 |

**适用场景**：需要快速执行单个操作时，命令面板比打开面板再点击更快捷。

### 日记格式

插件通过解析日记中的任务行来计算积分。

**任务行格式**：
```
- [ ] 任务名称 - 积分
- [x] 任务名称 - 积分
```

- `[ ]` 表示未完成，不计入积分
- `[x]` 表示已完成，计入积分
- 积分必须写在任务名称后面，用 `-` 分隔

**完整示例**：
```markdown
★ **今日三件事**（优先完成，上限300）

- [x] 完成项目报告 - 100
- [x] 学习新技能 - 100
- [ ] 锻炼身体 - 100

● **习惯打卡**

- [x] 早起 - 50
- [x] 运动 - 50
- [ ] 阅读 - 50

● **额外任务**（完成最多2项，上限100）

- [x] 额外任务A - 50
- [ ] 额外任务B - 50

⏰ **专注番茄钟**（每个50积分，最多300）

- [x] 完成 ⏱️
- [x] 完成 ⏱️
- [ ] 完成 ⏱️
```

**注意事项**：
- 区分大小写，`[X]` 不会被识别为已完成
- 积分数字必须紧跟在 `-` 后面
- 番茄钟不写积分，默认按配置计算

### 系统设置

进入**设置 → Supreme Player**，解锁后可配置：

#### 解锁机制

1. 连续点击**「🔒 点击解锁」**按钮 42 次
2. 解锁后按钮变为**「🔓 已解锁」**
3. 30 分钟后自动上锁
4. 上锁需连点 3 次确认

#### 基础设置

| 配置项 | 说明 |
|--------|------|
| 玩家名称 | 显示在状态栏悬浮面板中 |
| 数据存储位置 | 账户数据文件路径（默认自动检测） |
| 模板文件位置 | 每日笔记模板路径 |
| 更新日记模板 | 根据当前配置重新生成模板 |

#### 每日任务配置

配置每日任务的数量和积分：

- **每日三件事**：任务数量（默认 3）、每任务积分（默认 100）
- **习惯打卡**：可添加/删除习惯，设置每个习惯的积分
- **额外任务**：计分数量上限（默认 2）、每任务积分（默认 50）
- **番茄钟**：计分数量上限（默认 6）、每个积分（默认 50）

#### 完美打卡配置

- **奖励类型**：商城商品 / 特殊商品
- **商城商品**：选择商城中已有的商品作为奖励
- **特殊商品**：自定义只能通过完美打卡获得的商品
- **祝福语**：自定义完美打卡时的标题和祝福内容

#### 商品管理

| 操作 | 说明 |
|------|------|
| 添加商品 | 创建新的商城商品，配置名称、描述、价格、效果 |
| 编辑商品 | 修改已有商品的属性 |
| 删除商品 | 移除不需要的商品 |
| 导出配置 | 将商品配置导出为 JSON 文件 |
| 导入配置 | 从 JSON 文件导入商品配置 |

**商品效果类型**：
- 获得愿星（指定数量）
- 获得等级（指定数量）
- 随机愿星（设置最小/最大值）
- 获得 Buff（自定义名称、描述、持续时间）
- 获得积分（指定数量）
- 清除所有 Buff

#### 等级管理

- 查看所有等级配置
- 编辑等级称号、能力、阶段、颜色
- 添加新等级
- 删除等级

#### 货币管理

- 查看所有货币及当前持有量
- 编辑自定义货币的获取间隔和颜色
- 添加新货币类型
- 删除自定义货币

### 数据管理

#### 数据备份

1. 点击**「📤 导出」**下载 JSON 备份文件
2. 文件包含：账户数据、配置、商城配置

#### 数据恢复

1. 点击**「📥 导入」**选择备份文件
2. 确认后覆盖当前数据
3. 重新加载插件生效

#### 数据存储位置

| 文件 | 路径 | 内容 |
|------|------|------|
| 账户数据 | `Notes/DailyNotes/supreme-player-data.json` | 积分、等级、背包等 |
| 插件配置 | `.obsidian/plugins/supreme-player/config.json` | 等级、货币、任务配置 |
| 商城配置 | `.obsidian/plugins/supreme-player/shop-config.json` | 商品列表 |

### 模板管理

#### 更新模板

设置中点击**「📝 更新模板」**，根据当前配置重新生成日记模板。

#### 模板位置

默认自动检测每日笔记文件夹中的模板。可在设置中手动指定。

#### 模板内容

模板包含以下占位符：
- `{{date}}` - 自动替换为当前日期
- 任务行会根据配置自动填充数量和积分

## 常见问题

### Q: 打卡后积分没有增加？

检查日记格式，任务行格式为 `- [x] 任务名 - 积分`。

### Q: 如何获取更多愿星？

- 每 100 积分自动获得 1 颗
- 商城购买愿星充能器
- 完美打卡使用超级大钻石

### Q: 商品无法购买？

检查商城顶部显示的道具卡数量是否足够。

### Q: 悬浮面板不显示？

将鼠标移到状态栏文字上方，等待约 0.5 秒后弹出。
