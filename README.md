# Supreme Player（超越玩家系统）

一款基于游戏化设计的 Obsidian 自律习惯养成插件

## 简介

Supreme Player 将自律行为转化为游戏体验，通过积分、等级、许愿池、道具商城和背包系统，帮助你建立正向的行为反馈循环。

**核心特点**：
- 🎮 游戏化设计，让自律变得有趣
- ⚡ 状态栏一键操作，无需记忆命令
- 🌟 许愿池系统，愿望可视化
- 🎴 道具商城，即时奖励激励

## 功能概览

### 等级系统

| 等级 | 称号 | 能力 | 阶段 |
|------|------|------|------|
| 0-5 | ⭐ 星语者 | 扭曲现实 | 术士 |
| 6-10 | 🔮 织梦师 | 覆写真实 | 人神 |
| 11-15 | ⚛️ 创造主 | 造物创生 | 伪神 |
| 16-20 | 🌏 掌界者 | 掌控世界 | 真神 |
| 21-25 | 🌌 元星神 | 指尖寰宇 | 星神 |
| 26+ | ♾️ 无极尊 | 无穷界限 | ■■■■ |

### 货币与道具

- **愿星 ⭐**：用于许愿池扰动世界线
- **稀有道具卡 🎴**：购买稀有品质商品
- **传奇道具卡 🌠**：购买传奇品质商品

### 商城商品

| 商品 | 价格 | 效果 |
|------|------|------|
| 愿星充能器 ✨ | 1 稀有卡 | 获得 5 颗愿星 |
| 等级跃迁符 💫 | 1 传奇卡 | 等级 +1 |
| 神秘福袋 🎁 | 1 稀有卡 | 随机 1-10 颗愿星 |
| 牛奶 🥛 | 1 稀有卡 | 清除所有 Buff |

## 快速开始

### 1. 启用插件

在 Obsidian 设置 → 社区插件 中启用 Supreme Player。

### 2. 使用插件

将鼠标悬浮在底部状态栏上，弹出快捷面板：

```
┌─────────────────────────────┐
│ 👤 玩家 | ⭐ Lv.0 星语者    │
├─────────────────────────────┤
│ 📊 积分        850          │
│ 📈 距下一级    4150         │
├─────────────────────────────┤
│ ⭐ 愿星           8         │
│ 🎴 稀有道具卡     1         │
├─────────────────────────────┤
│ [⛲ 许愿] [🎴 商城] [📦 背包] │
└─────────────────────────────┘
```

点击按钮即可进入对应功能，或点击状态栏进入玩家面板。

### 3. 每日打卡

点击状态栏「📊 积分」打开玩家面板 → 点击「📅 每日打卡」。

## 进阶用法

### 命令面板

按 `Ctrl+P` 可直接执行命令：

| 命令 | 功能 |
|------|------|
| 每日打卡 | 执行今日打卡 |
| 查看当前状态 | 打开玩家面板 |
| 同步账户信息 | 同步数据到当前文件 |
| 许下愿望 | 创建新愿望 |
| 查看许愿池 | 管理愿望 |
| 打开商城 | 浏览购买商品 |
| 查看背包 | 使用背包物品 |

### 日记格式

任务行格式：`- [x] 任务名称 - 积分`

```markdown
★ **今日三件事**

- [x] 完成项目报告 - 100
- [ ] 学习新技能 - 100

● **习惯打卡**

- [x] 早起 - 50
- [x] 运动 - 50
```

### 系统设置

进入设置 → Supreme Player，解锁后可配置：

- 玩家名称
- 每日任务数量和积分
- 完美打卡奖励
- 商品管理
- 等级称号
- 货币类型
- 数据导入/导出
